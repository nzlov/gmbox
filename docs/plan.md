# 实施计划

## 项目目标

- 使用 `Go + Gin + Vue3` 实现单管理员电子邮件客户端。
- 使用 `embed` 嵌入前端静态资源。
- 使用 `github.com/nzlov/utils/db` 作为数据库框架。
- 默认支持 `sqlite`，同时兼容 `postgres` 与 `mysql`。
- 支持 `IMAP`、`SMTP`、`POP3`，其中 `POP3` 仅做收件。
- 支持多邮箱账户、聚合列表、单邮箱查看。
- 使用 Go 协程处理多邮箱并发同步。
- 使用 `config.yaml`，支持默认值回落与环境变量强覆盖。
- 进入网站需要登录；默认管理员从配置导入数据库，后续仅校验数据库密码。
- 使用 `github.com/robfig/cron/v3` 实现定时同步调度。
- 提供 Docker 部署能力。
