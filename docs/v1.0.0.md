# 1.0.0

## 侧边栏调整

聚合收件箱改名为聚合信息

侧边栏显示顺序调整：

退出登录放到最下面

例如：

```
聚合信息
写信
邮箱管理

<spac />

退出登录
```

## 聚合信息页主区域调整

### 布局调整

- 不再显示统计信息

- 左侧显示邮箱和文件夹列表

- 右侧邮件列表

### 左侧显示邮箱和文件夹列表

- 顶部是邮箱搜索，支持下拉选择。 如果情况显示全部邮箱的数据。
- 下面是文件夹
- 搜索邮箱后，文件夹列表更新，邮件列表更新

### 邮件列表

- 顶部邮件搜索，强关联左侧选择。

- 然后是邮件列表，使用时间倒序显示，带有分页， 支持分页大小调整。

## 邮箱管理

- 使用表格显示添加的邮箱列表
- 添加修改邮箱使用弹窗。
- imap和pop3互斥，只显示选择所需要的输入内容。
- 可多选启用、禁用、同步、删除、测试等。

- 添加按钮显示在表格上方。
